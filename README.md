# Personal portfolio and blog

Made with

- [Hapi][3]
- [Gatsby][4]
- [React][5]

## Development

- `npm run dev`
  - runs Gatbsy server in developer mode
  - watches and builds css files (imported into Gatsby)
  - watches tests
  - watches [lighthouse-ci][6]

- `npm run deploy`
  - runs tests
  - pushes to [Clever cloud][1]
  - pushes to [Github][2]

- `npm run dry-run` (to check deployment)
  - builds blog, css
  - starts server
